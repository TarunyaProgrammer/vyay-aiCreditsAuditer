# Key Performance Indicators (KPIs) — Vyay

We focus on "Value-First" metrics.

## 🚀 Primary North Star Metric
- **Total Savings Identified ($)**: The aggregate amount of spend identified as "leaking" across all users.

## 📈 Secondary Metrics
1. **Audit Completion Rate**: % of users who start the form and reach the result page.
2. **Time-to-Value (TTV)**: Average seconds from landing to first recommendation. (Goal: < 60s).
3. **Virality Coefficient (k)**: Number of new audits generated per share of a result URL.
4. **Tools per Audit**: Average number of tools users are auditing (indicates market depth).

## 📊 Technical Metrics
1. **Gemini Latency**: P99 response time for AI-assisted analysis.
2. **Error Rate**: % of failed audit calculations.
3. **Page Load Speed**: LCP (Largest Contentful Paint) for the landing page.
