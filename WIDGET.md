# Vyay Embeddable Widget

The Vyay widget allows you to integrate a simplified AI spend audit call-to-action on any website, blog, or internal portal.

## 1. Quick Integration

Add the following code to your HTML where you want the widget to appear:

```html
<!-- Widget Container -->
<div id="vyay-audit-widget"></div>

<!-- Widget Script -->
<script src="https://vyay.ai/widget.js" defer></script>
```

## 2. Widget Features
- **Lightweight**: < 2KB payload.
- **Zero Dependencies**: Pure Vanilla JavaScript.
- **Responsive**: Adapts to sidebar or main content widths.
- **Branding**: Implements the Vyay "Calm Engineering" aesthetic.

## 3. How it Works
The script identifies the `vyay-audit-widget` ID, injects a scoped shadow-style block to prevent CSS collisions, and renders a high-conversion CTA that links directly to the Vyay audit engine.

## 4. Customization
The widget is designed to be "Restrained" by default. If you need a custom version (e.g. for a dark mode site), please contact the engineering team.
